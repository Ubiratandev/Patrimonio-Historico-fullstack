# 🏛️ Patrimônio Cultural de Barra do Piraí

Plataforma GovTech desenvolvida para catalogar, preservar e democratizar o acesso aos bens históricos e culturais do município de Barra do Piraí - RJ.

O projeto foi patrocinado pela Lei de Incentivo à Cultura Aldir Blanc (PNAB) e criado com o objetivo de centralizar informações históricas do município em uma plataforma digital acessível, moderna e escalável.

---

# 🌎 Sobre o projeto

A plataforma reúne:

- mais de 100 textos históricos
- mais de 380 imagens catalogadas
- conteúdos produzidos por mais de 10 profissionais de diferentes áreas
- painel administrativo colaborativo
- sistema de moderação e publicação de conteúdo

Todo o sistema foi desenvolvido do zero, incluindo:

- arquitetura da aplicação
- modelagem de banco de dados
- escolha de stack
- desenvolvimento frontend e backend
- design das páginas
- sistema administrativo
- fluxo editorial

---

# ✨ Funcionalidades

## 📚 Catálogo histórico e cultural

- visualização de patrimônios históricos
- páginas detalhadas para cada item catalogado
- galeria de imagens
- organização e categorização dos conteúdos
- navegação responsiva

---

## 🛠️ Painel administrativo

Autores e pesquisadores podem:

- criar novos textos
- editar conteúdos existentes
- atualizar imagens
- gerenciar conteúdos históricos

---

## 🔎 Sistema de revisão editorial

O projeto possui um sistema de aprovação inspirado em Pull Requests.

### Fluxo:

1. autores realizam alterações
2. alterações ficam pendentes
3. super administrador revisa
4. conteúdo aprovado é publicado em produção

Esse sistema garante:

- segurança editorial
- controle de alterações
- preservação da integridade histórica
- fluxo colaborativo entre pesquisadores
